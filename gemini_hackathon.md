**Gemini 3 Hackathon — summary**

An **in-person Singapore hackathon** challenging teams (≤4) to build **ambitious, production-grade AI applications** using **Gemini 3**, **Google AI Studio**, and **Antigravity**. Submissions must go beyond demos to showcase **state-of-the-art reasoning, multimodality, agentic workflows, reliable tool use, or real-time voice/vision interactions**.

**Judging focus:** technical ambition, novel use of Gemini APIs, robustness, and end-to-end execution.

**Tracks:**

1. Infinite context & multimodal reasoning
2. Agentic coding (10× engineer)
3. Enterprise orchestration & autonomous actions
4. Generative media (audio/video/music)
5. Deep research & knowledge synthesis
6. Real-time multimodal (voice & vision)

**Flow:** 1-day hack → code freeze → video + GitHub submission → top-6 live demos.

**Prizes:** **100k Gemini API credits** + **founders call**.
